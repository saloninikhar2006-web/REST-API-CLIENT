DESCRIPTION:

This project demonstrates the development of a Java-based REST API client that fetches real-time data from a public API and displays it in a structured format.

The application sends an HTTP GET request to a weather API and retrieves data in JSON format. It then parses the JSON response to extract relevant information such as temperature, humidity, and weather conditions.

⚙️ Key Features

Sends HTTP requests using HttpURLConnection

Handles API responses efficiently

Parses JSON data using org.json library

Displays structured output in console

Includes error handling for failed requests

📦 Deliverable

A fully functional Java application that:

Connects to a public REST API

Handles HTTP requests and responses

Parses JSON data

Displays structured output in the console
